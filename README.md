# ⚡ Shortcuts

A simple and fast **web shortcuts launcher** that helps you open your favorite websites instantly.

---

## 🚀 What is this?

**Shortcuts** is a lightweight desktop app that lets you store and open web links quickly with one click.

Instead of typing URLs every time, just click your saved shortcut and go.

---

## ✨ Features

- 🌐 Open websites instantly
- ⚡ Super fast shortcut access
- 📌 Save your favorite links
- 🖥️ Clean and minimal interface
- 💡 Lightweight Electron app

## 🛠️ How it works

- You add a website link (example: YouTube, GitHub, etc.)
- The app stores it as a shortcut
- Click the shortcut → it opens in your browser instantly
